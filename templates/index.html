Ghotona Chitro - Event Detection API

Welcome to the Ghotona Chitro API. This API allows you to detect events in Bangla.

ঘটনা চিত্র এপিআই-এ স্বাগতম। এই এপিআই আপনাকে বাংলায় ইভেন্ট সনাক্ত করতে সাহায্য করে।

API Endpoints

Here are some of the endpoints you can use:

এখানে কিছু এন্ডপয়েন্ট দেওয়া হল যা আপনি ব্যবহার করতে পারেন:

Usage

ব্যবহার

To use this API, you need to send a GET/POST request to the appropriate endpoint.

এই এপিআই ব্যবহার করতে হলে, আপনাকে উপযুক্ত এন্ডপয়েন্টে একটি GET/POST অনুরোধ পাঠাতে হবে।
